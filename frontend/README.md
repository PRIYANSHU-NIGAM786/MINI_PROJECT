# 🌐 Roamify - Travel Itinerary Planner

Roamify is a full-stack MERN web application designed for seamless travel planning. Users can register, log in securely, and create customized day-by-day itineraries with integrated data filtering specific to each user.

## 🚀 Live Project Preview

![Project Dashboard](./screenshot.png)

## 🛠️ Features
- **User Authentication:** Robust secure signup and login flow.
- **Route Guards:** Page restriction on dashboard and itinerary generation pages for unauthenticated visitors.
- **Dynamic Scheduling:** Day-by-day planning fields dynamically render based on user-selected trip duration.
- **Personalized Dashboard:** Fetch, update, and cancel trip itineraries specific to the logged-in account.

## 💻 Tech Stack
- **Frontend:** Next.js, Formik, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
